Professional

These are projects I have done for work.

TACC — UI Agent

A multi-agent pipeline that ingests datasets from disk, summarizes them with tooling plus an LLM, plans a small dashboard app, and emits a runnable React/Vite project—fast handoffs enforced with Pydantic schemas for a Frontera-hosted research UI workflow.

github.com/kscnl1n/TACC-UI-Agent

  • Python
  • LangChain
  • Pydantic
  • React
  • LLM orchestration

Pacific Northwest National Lab — Code regression analysis

An ML-style pipeline that ties HPC benchmark behavior back to source structure: logs become text, CodeBERT embeddings land in vector space, and cosine-style similarity ranks what changed when performance regresses—supporting reproducible triage for complex scientific software. More detail is in Paper — Performance Regressions in Lamellar.

  • Python
  • PyTorch / Transformers
  • ChromaDB
  • NumPy
  • ML pipelines

TACC — AI-Spy

A generative-AI surface for research storytelling: visualize outputs with Stable Diffusion-style generation, keep fine-tunes practical with hardware-aware LoRA workflows, and ship the stack in containers so collaborators can iterate without rebuilding the world by hand. Supporting scripts and notes live alongside copy_of_paper2abstract.py.

  • Python
  • Stable Diffusion
  • LoRA fine-tuning
  • Docker
  • Generative AI

TACC — Project Astra

End-to-end data work for continuous methane monitoring: ingest IoT and emissions feeds, normalize and land them in SQL for analysis, then serve operational insight through dashboards (Plotly Dash, Flask, Tableau) so detection and quantification stories stay close to the data. Learn more about the program at projectastra.energy.

  • Python
  • SQL / ETL
  • IoT data
  • Dash / Flask
  • Tableau

UT Austin — YOLO With Caution

A GPU-accelerated YOLOv8 vision study on a large multispectral corpus—tuning detection quality while explicitly stress-testing demographic bias so “accuracy” is not the only headline. Artifacts include the CIR4SC poster and the write-up YOLO With Caution.

  • PyTorch
  • YOLOv8
  • Computer vision
  • GPU / HPC
  • Fairness analysis
